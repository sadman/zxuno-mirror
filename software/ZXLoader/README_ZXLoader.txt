MODO DE USO

  Para ver ficheros en el disco local, copiarlos primero a data\misc\zxloader\local\
  
  Para abrir ficheros con CargandoLeches hay que preparar el script TAP2WAVZXLauncher.bat (TAP2WAVZXLauncher.sh en Linux)
  Por defecto viene configurado para ejecutar el VLC, y hay una espera de 5 segundos que se puede editar o eliminar.
  
  Para abrir el programa, ejecutar ZXLoader.bat (ZXLoader.sh en Linux)

  Al hacer un click sobre un directorio se abrira. Puede tardar un poco si es de WOS.
  
  Al hacer un click sobre un fichero se abrira con el script indicado en la pestaña Options. Se ejecutara la accion
  pertinente si es un .tap o un .z80, y si es un zip se descomprimira primero.
  
  Cuando se esta lanzando un programa el titulo de la ventana cambia a "Running process", y cuando esta disponible pone "Ready".
  Puedes seguir navegando y abriendo directorios, incluso de WOS, pero no podras lanzar otro archivo hasta que termine el anterior.
